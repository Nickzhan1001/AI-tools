# Anthropic Skills 功能一覽

**來源**：anthropics/skills（透過 OpenSkills 安裝）

---

## 1. algorithmic-art

**來源**：anthropics/skills

**演算法藝術創作與 p5.js**：結合種子隨機與參數探索，產出可重現的程式藝術（粒子、流場、幾何等），並避免抄襲既有作品。

## 2. brand-guidelines

**來源**：anthropics/skills

**Anthropic 官方品牌識別與風格**：色彩、字體與視覺規範，適用於需符合 Anthropic 品牌的使用情境（簡報、文件、對外素材）。

## 3. canvas-design

**來源**：anthropics/skills

**視覺設計與產出流程**：結合視覺哲學與實作步驟（構圖、色彩、排版、輸出格式），產出海報、插圖或品牌視覺等靜態設計。

## 4. doc-coauthoring

**來源**：anthropics/skills

**文件共同撰寫流程與引導**：從大綱、草稿、修訂到定稿的協作步驟，適合規格、提案或技術文件與 agent 協作撰寫。

## 5. docx

**來源**：anthropics/skills

**Word 文件建立 / 編輯 / 分析**：引導產出或修改 .docx（段落、標題、清單、表格、追蹤修訂），並可擷取與分析文件結構與文字。

## 6. frontend-design

**來源**：anthropics/skills

**前端介面設計與實作**：引導產出有辨識度、可上線的前端（元件/頁面/應用），強調設計思維、排版、色彩、動效等美學細節，避免通用「AI 感」版面與字體。

## 7. internal-comms

**來源**：anthropics/skills

**內部溝通文件撰寫**：3P（Progress/Plans/Problems）、週報、公告、FAQ 等格式與要點，產出結構清晰、易讀的內部更新。

## 8. mcp-builder

**來源**：anthropics/skills

**MCP（Model Context Protocol）伺服器設計與實作**：涵蓋工具定義、資源、傳輸與錯誤處理，適合打造或整合 MCP 伺服器供 agent 使用。

## 9. pdf

**來源**：anthropics/skills

**PDF 處理指引**：涵蓋讀取、合併、拆分、表單填寫與基本生成，讓 agent 能依需求操作 PDF 檔案。

## 10. pptx

**來源**：anthropics/skills

**PowerPoint 建立 / 編輯 / 分析**：引導以程式或描述產出、修改 .pptx（投影片、版型、圖表、備註），並可分析既有簡報結構與內容。

## 11. skill-creator

**來源**：anthropics/skills

**建立與設計 Skills 的方法與規範**：說明如何撰寫 SKILL.md、觸發條件、步驟與資源，讓 agent 能正確載入與執行自訂或第三方技能，適合要發布或維護 agent 技能的人。

## 12. slack-gif-creator

**來源**：anthropics/skills

**製作適合 Slack 的 GIF**：尺寸、幀率、檔案大小與動畫概念建議，產出在 Slack 中顯示良好的動態圖。

## 13. theme-factory

**來源**：anthropics/skills

**字體 / 配色主題庫與套用流程**：提供預設主題與變數結構，讓 agent 能快速套用或微調專案的字型與色彩系統。

## 14. web-artifacts-builder

**來源**：anthropics/skills

**建立 Claude Web Artifacts 的流程與規範**：引導產出可在 Claude 中預覽的 HTML/JS/CSS 成品，包含結構、互動與部署考量。

## 15. webapp-testing

**來源**：anthropics/skills

**以 Playwright 測試本機 Web 應用**：涵蓋撰寫與執行 E2E 測試、選擇器策略、截圖與除錯，適合對本機或 staging 網頁進行自動化測試。

## 16. xlsx

**來源**：anthropics/skills

**Excel 試算表處理**：涵蓋讀寫 .xlsx、公式、格式、錯誤防護與資料驗證，適合報表、資料匯入匯出與建模。
