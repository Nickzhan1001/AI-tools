# Agent Skill 排行與簡介（1–100）

本文件依 [The Agent Skills Directory（skills.sh）](https://skills.sh/) 的統計與說明整理，提供前 100 名技能的**詳細簡介**與安裝來源，方便挑選與安裝。

- **安裝方式**：`npx skills add <owner/repo>` 或指定 skill：`npx skills add https://github.com/<owner>/<repo> --skill <skill-name>`
- **資料來源**：<https://skills.sh/>（排行與週安裝數以站上為準）
