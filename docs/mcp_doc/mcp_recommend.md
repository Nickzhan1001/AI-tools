# MCP 推薦清單

**資料來源**：<https://smithery.ai/>

本文件整理可搭配 Cursor、Claude、Windsurf 等 AI 編程工具的 **MCP（Model Context Protocol）伺服器**推薦清單，含來源連結與簡短功能說明，方便挑選與安裝。清單來自 [Smithery](https://smithery.ai/)，可至該站查詢更多 MCP 與安裝方式。

---

## Browserbase

**來源**：https://smithery.ai/server/browserbasehq/mcp-browserbase

**雲端瀏覽器自動化**：在雲端執行 headless/headed 瀏覽器，供 agent 進行導覽、點擊、填表、截圖等操作，適合網頁爬取、E2E 測試或需真實瀏覽環境的自動化任務。

## Context7

**來源**：https://smithery.ai/server/upstash/context7-mcp

**即時文件與程式碼範例**：可查詢任意函式庫、框架的最新文件與程式碼範例，適合開發時需要查證 API 用法、版本差異或複製範例的情境。

## ref

**來源**：https://smithery.ai/server/ref-tools/ref-tools-mcp

**文件搜尋與讀取**：提供 `ref_search_documentation`、`ref_read_url` 等工具，在函式庫、框架或 API 文件內搜尋並讀取內容，適合撰寫或除錯程式時查閱官方文件。

## Exa Search

**來源**：https://smithery.ai/server/exa

**語意化網頁搜尋與研究**：提供即時網頁搜尋、URL 內容擷取與公司/組織研究，工具包含 `web_search_exa`（可設 auto/fast/deep）、`company_research_exa`（營運、新聞、產業分析）、`get_code_context_exa`（函式庫/API/SDK 最新脈絡，減少 AI 幻覺）。適合研究、競品分析或為 agent 補充即時網路與程式脈絡。
